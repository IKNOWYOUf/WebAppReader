# WebAPP

> 基于 html5 和 Vue.js 的手机小说阅读器 
>> [https://carolcoral.github.io/WebAPP](https://carolcoral.github.io/WebAPP)

## 技术结构：
* 前端采用 Html5 和 Vue.js 以及 Koa 框架进行开发；
* 模拟后台数据使用 EJS 模版的路由功能以及 node.js；
